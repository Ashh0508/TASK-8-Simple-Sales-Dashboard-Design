# 📊 Simple Sales Dashboard Design

## 📌 Project Overview

This project presents an interactive Sales Dashboard developed using **Microsoft Power BI** to analyze sales performance across different regions, product categories, and time periods. The dashboard enables business stakeholders to monitor key performance indicators (KPIs), identify sales trends, and compare regional and category-wise performance through interactive visualizations.

---

# 🎯 Objectives

* Build an interactive sales dashboard.
* Visualize sales trends over time.
* Compare sales performance across regions.
* Analyze category-wise sales contribution.
* Enable users to filter data using slicers.
* Present key business insights for decision-making.

---

# 🛠️ Tools Used

* Microsoft Power BI
* Microsoft Excel / CSV Dataset
* DAX (Data Analysis Expressions)

---

# 📂 Dataset

The dashboard is built using a Superstore Sales dataset containing:

* Order Date
* Order ID
* Region
* Category
* Sales
* Profit
* Customer Information
* Product Information

---

# 📈 Dashboard Features

### KPI Cards

* 💰 Total Sales
* 📈 Total Profit
* 📦 Total Orders

### Visualizations

* 📈 Monthly Sales Trend (Line Chart)
* 📊 Regional Sales Performance (Bar Chart)
* 🍩 Sales Contribution by Category (Donut Chart)

### Interactive Filters

* 📍 Region Slicer

---

# 📊 Key Insights

* The West region recorded the highest sales performance.
* Technology contributed the largest share of overall sales.
* Sales fluctuated across different months, indicating seasonal demand patterns.
* Interactive filters allow users to analyze performance by region.
